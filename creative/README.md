# CORO PROJECT Creative Support

Static landing page for `https://coroproject.jp/creative/`.

## Files

- `index.html`: published page markup and inline interaction script
- `src/index.css`: page styles loaded directly from `index.html`
- `vite.config.ts`: local Vite dev server configuration

## Local Commands

```bash
npm install
npm run dev
npm run build
```

The public Apache route can serve `index.html` directly; Vite is only for local development and build checks.
